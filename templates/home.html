⚡ FCFS

First Come First Serve

Type: Non-Preemptive

Difficulty: Easy

Executes processes in arrival order

Definition: Executes processes in arrival order.

P213

🎯 SJF

Shortest Job First

Type: Non-Preemptive

Difficulty: Medium

Executes process with smallest burst time

Definition: Smallest burst time executes first.

PATBT
P105
P213
P322

⚡ SRTF

Shortest Remaining Time First

Type: Preemptive

Difficulty: Hard

Shortest remaining time executes first

Definition: Preemptive SJF.

P1 starts → P2 arrives → P2 runs → P1 resumes
