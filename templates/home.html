🔥 CPU Scheduler Simulator

Interactive visualization of CPU scheduling algorithms with animations, metrics, and dynamic simulation.

🧠 Algorithms

FCFS

First come, first serve. Simple but can cause long waiting.

SJF

Shortest job first. Minimizes waiting time.

SRTF

Preemptive version of SJF. Always picks shortest remaining job.

Round Robin

Each process gets equal CPU time slice.

Priority

Higher priority process executes first.
