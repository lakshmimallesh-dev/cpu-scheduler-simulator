# 🚀 CPU Scheduling Simulator

An interactive web-based CPU Scheduling Simulator that helps visualize how different scheduling algorithms work in real-time.

---

## 🌟 Features

* 📘 **Algorithm Explanation**

  * FCFS (First Come First Serve)
  * SJF (Shortest Job First)
  * SRTF (Shortest Remaining Time First)
  * Round Robin
  * Priority Scheduling

* ⚡ **Real-Time Simulation**

  * Visual representation of process execution
  * Gantt chart generation
  * Calculation of:

    * Waiting Time (WT)
    * Turnaround Time (TAT)

* 🤖 **AI Chatbot Support**

  * Ask doubts related to CPU scheduling
  * Get explanations with examples

* 🎨 **Modern UI**

  * Smooth animations
  * Interactive dashboard
  * Floating chatbot interface

---
